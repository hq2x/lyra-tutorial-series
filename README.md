## 🚀 快速开始

### 前置要求

- ✅ Unreal Engine 5.1 或更高版本
- ✅ Visual Studio 2022 (Windows) 或 Xcode (Mac)
- ✅ 基础的 C++ 和 UE 开发经验
- ✅ Git 基础知识

### 获取 Lyra 项目

```bash
# 方法 1: 通过 Epic Games Launcher 下载（推荐）
# 在 Launcher 的 "Learn" 标签下找到 Lyra Sample Game

# 方法 2: 从 GitHub 源码编译（需要 Epic 账号关联）
git clone --filter=blob:none --sparse https://github.com/EpicGames/UnrealEngine.git
cd UnrealEngine
git sparse-checkout set Samples/Games/Lyra
git checkout ue5-main
```

### 编译项目

```bash
# Windows
cd Samples/Games/Lyra
.\Build\BatchFiles\GenerateProjectFiles.bat
# 打开 Lyra.sln，编译 Development Editor 配置

# Mac/Linux
./Build/BatchFiles/GenerateProjectFiles.sh
# 编译项目
```

## 📊 技术栈总览

### 引擎核心系统

| 技术 | 说明 | 难度 |
|------|------|------|
| **Gameplay Ability System (GAS)** | Epic 官方技能系统 | ⭐⭐⭐⭐⭐ |
| **Enhanced Input System** | UE5 新一代输入系统 | ⭐⭐⭐ |
| **Modular Gameplay** | 模块化游戏框架 | ⭐⭐⭐⭐ |
| **Game Features** | 插件化内容系统 | ⭐⭐⭐⭐ |
| **Common UI** | 跨平台 UI 框架 | ⭐⭐⭐ |
| **Replication Graph** | 网络优化系统 | ⭐⭐⭐⭐⭐ |
| **Data Registry** | 数据资产管理 | ⭐⭐⭐ |

### Lyra 自定义系统

- Experience System (体验系统)
- Team System (队伍系统)
- Equipment & Weapon System (装备与武器)
- Inventory System (背包系统)
- Game Phase System (游戏阶段)
- UI Extension System (UI 扩展)
